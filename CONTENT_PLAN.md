### 全局组件规则

- 最大内容宽度：`max-w-[1200px]`，桌面端大量留白
- 章节标记：`[01]` `[02]` `[03]` `[04]`，使用 `text-accent-blue/60 font-mono text-sm`
- 分割线：`border-b border-border`，1px
- 悬停卡片：`hover:bg-surface-hover transition-colors duration-300`
- 按钮主样式：`bg-accent-blue text-white px-8 py-4 text-base font-medium`（保留直角，不加圆角——BrowserOS 语法）
- 按钮次样式：`border border-border px-8 py-4 text-base font-medium`

### Hero — 手语图片轮播 + 标题叠加

```
┌─────────────────────────────────────────────────────────┐
│ ┌───────────────────────────────────────────────────┐   │
│ │                                                   │   │
│ │   [ 手语沟通照片 1 ]  [ 照片 2 ]  [ 照片 3 ]  [ 照片 4 ]│   │
│ │   ← 自动轮播，每张 4 秒，淡入淡出过渡 →              │   │
│ │                                                   │   │
│ └───────────────────────────────────────────────────┘   │
│                                                         │
│         别让声音                                          │
│         限制了你的表达。                                    │
│                                                         │
│         赋语鸭是一个 AI 沟通代理——                          │
│         理解你的关键词、场景和情绪，                          │
│         把碎片化输入转成自然、有尊严的表达。                    │
│                                                         │
│         [了解赋语鸭 ↓]                                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**设计要点**：
- 轮播区：`100vh` 全屏高度，图片 `object-cover` 填充
- 每张图片上有半透明深色遮罩（`bg-black/30`），确保白色文字可读
- 图片过渡用 `opacity` 淡入淡出（性能最优），不用滑动
- 轮播指示器：底部居中的小圆点，当前激活为蓝色
- 文字层 absolute 定位叠在图片上方，居中
- 标题用思源宋体 Bold，`clamp(48px, 6vw, 96px)`，白色（`text-white` 或暖米色 `text-[#f4f1e6]`）
- 副标题 20px，白色半透明 `text-white/80`
- 「了解赋语鸭 ↓」按钮：透明底 + 白色边框，滚动到下一区块
- **关于手语照片**：使用 Unsplash 免费图库的手语/沟通主题照片 URL 作为占位（见下方「图片资源清单」），可替换为你自己的素材

## 全局动效系统

| 动效 | 实现 | 适用 |
|------|------|------|
| 轮播切换 | `opacity` 淡入淡出，`duration: 1s` | Hero 图片轮播 |
| 区块入场 | Framer Motion `fadeInUp`，`duration: 0.6` | 每个 section 进入视口 |
| 卡片悬停 | `scale: 1.02` + 阴影提升，`0.3s ease` | 功能卡片、场景卡片 |
| 数字滚动 | 0 → 目标值，`duration: 1.5s` | 信任指标栏 |
| 手风琴展开 | `height` 动画 + 箭头旋转 | FAQ |
| 导航显隐 | `translateY`，`duration: 0.3s` | Navbar |
| 横向滑动 | 鼠标拖拽 + 惯性，触控手势 | 人群卡片 |

**性能铁律**：所有动画只动 `transform` 和 `opacity`。`prefers-reduced-motion` 时静默所有动效。
